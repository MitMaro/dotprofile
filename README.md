Dot Profile
===========

Requirements
------------

### All OS Versions

* [t](http://stevelosh.com/projects/t/)
* [j2](https://github.com/rupa/j2)
* [tasks](https://github.com/MitMaro/tasks)

### Mac OSX

* [Homebrew](http://mxcl.github.com/homebrew/) with following packages installed
    * git
    * git-extras
    * git-utils
    * colordiff
    * bash-completion
    * coreutils
    * readline
